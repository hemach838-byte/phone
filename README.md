# 📱 PhoneShop UI

A modern and responsive phone e-commerce website built using HTML, CSS, and JavaScript.

This project allows users to browse phones, search products, filter by categories and brands, and view phone details in a clean user interface.

---

## ✨ Features

- 🔍 Search phones instantly
- 📂 Category filters
- 🏷️ Brand filters
- ❤️ Wishlist icon
- 🛒 Cart icon
- 📱 Responsive product cards
- 📖 View Details section
- 🎨 Modern UI design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
